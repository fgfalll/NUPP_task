# Монітор Завдань для Календаря НУПП

**Автор:** Петренко Тарас Сергійович
**Версія:** 2.3.0
**Рік:** 2025

Ця програма надає настільний інтерфейс для моніторингу завдань з календаря Національного університету "Полтавська політехніка імені Юрія Кондратюка".

## Функціонал

- **Управління завданнями:** Переглядайте активні завдання, відстежуйте прогрес та керуйте дедлайнами.
- **Робота з документами:** Завантажуйте та керуйте документами, пов'язаними із завданнями.
- **Обмін:** Діліться інформацією про завдання через Viber та Telegram із попередньо заповненими повідомленнями.
- **Архів:** Переглядайте виконені та прострочені завдання.
- **Безпека:** Захист паролем із можливістю відновлення доступу.

## Встановлення та розробка

1. **Вимоги:** Python 3.8+
2. **Клонуйте репозиторій:**
   ```bash
   git clone https://github.com/fgfalll/NUPP_task
   cd NUPP_task
   ```
3. **Налаштуйте середовище:**
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # Windows: .venv\Scripts\activate
   pip install -r requirements.txt
   ```
4. **ЗАМІНІТЬ SECRET KEY** (див. вище "Важливо")
5. **Запустіть програму:**
   ```bash
   python task_monitor.py
   ```

## Використання

### Перший запуск

1. Встановіть пароль для доступу до програми.
2. Збережіть ключ відновлення, який буде відображено.
3. Вкажіть відповіді на два запитання для відновлення (можна пропустити).

### Вхід до системи

Введіть свої облікові дані для календаря НУПП.

### Відновлення пароля

Якщо ви забули пароль, запустіть `PasswordRestore.exe` та використайте один із методів відновлення:

- **Ключ відновлення** - введіть збережений ключ
- **Запитання для працівників** - якщо налаштовані при першому запуску
- **Адміністративний файл** - перетягніть `.task_monitor_admin.key` файл (згенерований `AdminKeyGen.exe`)

### Генерація адміністративного файлу

> **Увага:** Адміністративний файл дозволяє відновити доступ БЕЗ знання поточного пароля.

Для створення адміністративного файлу:
1. Запустіть `AdminKeyGen.exe`
2. Програма автоматично згенерує `.task_monitor_admin.key` файл
3. Перетягніть цей файл у вікно `PasswordRestore.exe`

## Збірка у виконуваний файл

### Підготовка до збірки

1. **Встановіть PyInstaller:**
   ```bash
   pip install pyinstaller
   ```
2. **Зробіть зміни до secrets** (див. "Важливо" вище)
3. **Перевірте, що всі hardcoded секрети замінені**

### Збірка

```bash
python build.py
```

### Очищення

```bash
python build.py --clean
```

### Результат

Файли будуть у папці `dist/`:
- `TaskMonitor.exe` - основна програма
- `PasswordRestore.exe` - програма відновлення пароля
- `AdminKeyGen.exe` - генератор адміністративного файлу
