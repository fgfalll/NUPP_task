# Монітор Завдань для Календаря НУПП

**Автор:** Петренко Тарас Сергійович

Ця програма надає настільний інтерфейс для моніторингу завдань з календаря Національного університету "Полтавська політехніка імені Юрія Кондратюка".

## Функціонал

- **Управління завданнями:** Переглядайте активні завдання, відстежуйте прогрес та керуйте дедлайнами.
- **Робота з документами:** Завантажуйте та керуйте документами, пов'язаними із завданнями.
- **Обмін:** Діліться інформацією про завдання через Viber та Telegram із попередньо заповненими повідомленнями.
- **Архів:** Переглядайте виконані та прострочені завдання.
- **Безпека:** Безпечний доступ до програми із захистом паролем.

## Встановлення

1.  **Вимоги:** Python 3.8+
2.  **Клонуйте репозиторій:**
    ```bash
    git clone https://github.com/your-username/task-monitor.git
    cd task-monitor
    ```
3.  **Встановіть залежності:**
    ```bash
    pip install -r requirements.txt
    ```
4.  **Запустіть програму:**
    ```bash
    python task_monitor.py
    ```

## Використання

1.  **Перший запуск:** Встановіть пароль для програми.
2.  **Вхід:** Введіть свої облікові дані для календаря НУПП.
3.  **Керування завданнями:** Переглядайте, відстежуйте та діліться своїми завданнями.

## Збірка

Для створення виконуваного файлу (`.exe`) з програми, виконайте наступні кроки:

1.  **Встановіть `pyinstaller`:**
    ```bash
    pip install pyinstaller
    ```
2.  **Створіть іконку:** Переконайтеся, що у вас є файл `icon.ico` в кореневій папці проекту.
3.  **Запустіть скрипт збірки:**
    ```bash
    python build.py
    ```
    Після завершення, ви знайдете виконуваний файл у папці `dist`.
